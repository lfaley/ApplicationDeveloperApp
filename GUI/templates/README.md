# Template Directory Structure for New Project Creation Agent

This directory contains starter files for each supported framework. These files are copied into the new project directory during creation.

## Structure

- templates/
  - react/
    - README.md
    - .gitignore
    - App.js
  - next.js/
    - README.md
    - .gitignore
    - pages/index.js
  - node.js/
    - README.md
    - .gitignore
    - index.js
  - python/
    - README.md
    - .gitignore
    - main.py
  - c#/
    - README.md
    - .gitignore
    - Program.cs

## Notes
- Each README.md provides a basic project description.
- .gitignore files are tailored for each framework.
- Main source file is included for each language/framework.
- These are minimal starter files; users can customize templates as needed.
